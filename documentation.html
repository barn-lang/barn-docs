Building
Building for Linux will look like this:
git clone https://github.com/barn-lang/barn
cd barn
bash build.sh
barn --help
Or if you want you can download binary releases in our github repository. Getting Barn for Windows will look like this:
git clone https://github.com/barn-lang/barn
cd barn
./build.bat
./barn
If you have any type of errors with building or running barn compiler get help at our discord server.
Hello, World
After getting your barn compiler up and running maybe let start with creating the most easiest program that will just print "Hello, World!". Here's example code of program like that, you can write it by using puts function without importing std.ba library which makes it more lightweight, or using print/ println. We will show two of them, let's start with the lightweight one:
fun main() {
 puts("Hello, World!")
}
Now the other way of printing "Hello, World!" by using std (Standard Library). About this library is written more in the Standard Library chapter. Funfact you can contribute to this library and add functions that you want too! Anyways here is the example
@import "std.ba"

fun main() {
 println("Hello, World!")
}
We can see that in the first line we are import std.ba library, Next we are declaring a function named main which is the entry point of every program
Tip!
Without main function your program won't compile, 'cause the error that barn compiler is sending.
and next in the function body we are calling println which will print the "Hello, World" thing and go to the newline , puts function does it too so we don't need to write at the end of string \n which is a character indicating newline
Comments
In Barn there's two types of comments inline comments and multiline comments. Comments are these parts of code that compiler doesn't see you can put in them everything and that won't change your program work. Here is an example of it:
// This is an inline comment, it will end on newline
/*
 This is an multiline comment, it will end until we will write this
*/
Tip!
It is good and smart to comment your code in parts when it's little bit unreadable otherwise your code will be a total mess like Barn Compiler 😊
Operators
Operators are pretty important in every language, they do math, sometimes manipulate arrays and open/close functions bodies and arguments, in this chapter we will explain how operators works in Barn. There are actually 20 operators that work in barn
COMMA // , Does seperate arguments in functons
ARROW // -> After this symbol is type return of function
OPENPARENT // ( Opens arguments
CLOSEPARENT // ) Closes arguments
OPENBRACKET // { Open body function
CLOSEBRACKET // } Close body function
OPENBRACE // [ Open array
CLOSEBRACE // ] Open array
OROR // || Or
ANDAND // && And
GT // > Great than
GTE // >= Great than of equal
LT // < Less than
LTE // <= Less than of equal
ASN // = Assigment
EQ // == Equals
NEQ // != No equals
PLUS // + Plus
MINUS // - Minus
MUL // * Multiply
DIV // / Divide
MOD // % Modulus
BANG // ! Bang
PLUSASN // += Plus assigment
MINUSASN // -= Minus assigment
MULASN // *= Multiply assigment
DIVASN // /= Divide assigment
INCREMENTATION // ++ Inrementation of an value
DECREMENTATION // -- Decrementation of an value
