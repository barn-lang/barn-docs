Barn documentation

What is barn?
It's a simple programming language written in Go that is compiled to C. Syntax is very easy and complex, Barn can call C.
Barn is very optimilized and fast, it was written with idea to be the most fastes as he can and the most easiest to use as he can.

How to learn barn?
You can learn it by reading this documentation carefully

Community
We have own discord server where you can get help with barn

How to download barn

Dependeces
  • GoLang compiler
  • Bash (for unix)
  • GCC

MacOS & Linux
git clone https://github.com/barn-lang/barn
cd barn
bash build.sh
cp ./barn /bin/barn
barn # Should work fine

Windows
git clone https://github.com/barn-lang/barn
cd barn
./build.bat
./barn # Should work fine

Your first "Hello, World" in Barn

Create file and write to it this content
fun main() {
 puts("Hello, World!")
}
then compile it with command
barn ./hello_world.ba

Types

In barn there are 5 types:
  • string ("hello")
  • char ('a')
  • int (100)
  • float (4.3)
  • bool (true/false)

Functions

Function declaration
To declare a function you need to do this type of grammar
fun function_name() {
 /* function body */
}

Function return
To do function returing after () you need to give there -> /* data type */ (Data types)
fun get_age() -> int {
 /* function body */
 return 18
}

Function arguments
Argument function need to be provided between ()
fun add_two_numbers(int first_num, int second_num) -> int {
 /* function body */
 return first_num + second_num
}

Variables

To declare a variable you need to do this grammar
fun main() {
 let age: int = 18
}

Modify variable

To assign new variable you need to do this
fun main() {
 let age: int = 18
 age = 21
}

To add value to a variable you need to do this
fun main() {
 let i: int = 0
 i += 1 // i = 0 + 1
}

To minus value to a variable you need to do this
fun main() {
 let i: int = 1
 i -= 1 // i = 1 - 1
}

To multiply value to a variable you need to do this
fun main() {
 let i: int = 6
 i *= 2 // i = 6 * 2
}

To divide value to a variable you need to do this
fun main() {
 let i: int = 6
 i /= 2 // i = 6 / 2
}

If, elif, else

If statement
fun main() {
 if |1 == 1| {
  puts("1 == 1")
 }
}

Elif statement
fun main() {
 if |1 == 1| {
  puts("1 == 1")
 } elif |1 == 2| {
  puts("1 == 2")
 }
}

Else statement
fun main() {
 if |1 == 1| {
  puts("1 == 1")
 } elif |1 == 2| {
  puts("1 == 2")
 } else {
  puts("1 != 1 || 1 != 2")
 }
}

While

While statement
fun main() {
 let i: int = 0
 while |i < 5| {
  printnum(i)
  i++
 }
}

For

For statement
fun main() {
 for let i: int = 0; i < 5; i++ {
  printnum(i)
 }
}

Break

fun main() {
 for let i: int = 0; i < 5; i++ {
  if |i == 3| {
   puts("i == 3")
   break
  }
  printnum(i)
 }
}

Continue

fun main() {
 for let i: int = 0; i < 5; i++ {
  if |i == 3| {
   puts("i == 3")
   continue
  }
  printnum(i)
 }
}
