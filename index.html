How to learn?

Barn is extreamly easy and don't have any complex mechanics, but as every programming language you need to
write & learn to get fluent in it. The best source of learning is our Documentation where everything is shown very
well how to use Barn properly and use the Barn code style. Remember Barn is still in development and
everything can always change that's why our Documentation is divided in subpages that show every version
documentation
@import "std.ba"

fun main() {
 let is_barn_hard: auto = false // Of course it isn't hard
 if is_barn_hard == false {
  println("Barn is extreamly easyyy")
 } else {
  println("This print wouldn't show anytime soon, because barn is easy lol")
 }
}
