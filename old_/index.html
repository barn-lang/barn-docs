About us
Fast and easy programming!
Create everything you really like in a brand new, complex and easy syntax
programming language that is well documented. That's all Barn!

Look at our easy syntax
Barn syntax is very very easy, it's inspired with syntax languages like C, JavaScript, Go, Rust and other. Syntax is still in development, and we are constantly changing it!
@import "std.ba"

fun main() {
 println("Hello, World!")
}
How to learn?
Barn is extreamly easy and don't have any complex mechanics, but as every programming language you need to write & learn to get fluent in it. The best source of learning is our Documentation where everything is shown very well.
@import "std.ba"

fun main() {
 let is_barn_hard: auto = false // Of course it isn't hard
 if is_barn_hard == false {
  println("Barn is extreamly easyyy")
 } else {
  println("This print wouldn't show anytime soon, because barn is easy lol")
 }
}
100% Free and Open Source
Everything that is open source and free is better! Like Linus Torvald the Linux Kernel author said:
Software is like sex: it's better when it's free. ~ Linus Torvald
It will be pretty weird when an compiler wouldn't be free and globally avaliable. So everyone can build, run and code in barn 100% free and safe.
@import "std.ba"

fun main() {
 let open_source_software: auto = true
 if open_source_software == true {
  println("This software is cool!")
 } else {
  println("Nah we want the source lol")
 }
}
